| Coluna                        | Descrição                                          |
| ----------------------------- | -------------------------------------------------- |
| Transaction ID                | Identificador único da transação                   |
| Date                          | Data da transação (YYYY-MM-DD)                     |
| Account Type                  | Tipo da conta (Revenue, Expense, Asset, Liability) |
| Transaction Amount            | Valor bruto da transação                           |
| Cash Flow                     | Fluxo de caixa associado à transação               |
| Net Income                    | Resultado líquido                                  |
| Revenue                       | Receita gerada                                     |
| Expenditure                   | Despesa associada                                  |
| Profit Margin                 | Margem de lucro                                    |
| Debt-to-Equity Ratio          | Indicador de endividamento                         |
| Operating Expenses            | Custos operacionais                                |
| Gross Profit                  | Lucro bruto                                        |
| Transaction Volume            | Volume agregado de transações                      |
| Processing Time (seconds)     | Tempo de processamento                             |
| Accuracy Score                | Indicador de qualidade da transação                |
| Missing Data Indicator        | Indica presença de dados ausentes                  |
| Normalized Transaction Amount | Valor normalizado da transação                     |
| Transaction Outcome           | Resultado ( 1 Success / 0 Failure)                 |
